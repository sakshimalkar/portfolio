# Personal Portfolio Website

A modern, responsive personal portfolio website built using **React.js** and **Bootstrap**.  
This portfolio highlights my frontend development skills, real-world projects, and professional background in a clean and user-friendly layout.

## 🚀 Features
- Fully responsive design (mobile, tablet, desktop)
- Projects showcase with descriptions and videos
- Skills and courses section
- About Me and Contact sections
- Fast loading and optimized UI

## 🛠 Tech Stack
- React.js
- Bootstrap
- JavaScript (ES6+)
- HTML5
- CSS3
